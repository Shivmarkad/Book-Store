**Book Store App**

**To run this project locally follow these steps**

**1 : clone project**

```
git clone https://github.com/Shivmarkad/Book-Store.git
```
**2 : Switch Directory**
```
cd Book-Store
```

**3 : Install packages**

```
npm install
```

**4 : Create .env file**
```
touch .env
```

**5 : Copy .example_env to .env**

```
cp .example_env .env
```

**6: Update .env file with actual details**

**7: Run Project**

```
npm run dev
```
